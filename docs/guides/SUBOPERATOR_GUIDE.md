## Adding Customers

### Before Installation

**Checklist**:
1. Verify service available in area
2. Check customer eligibility
3. Confirm package pricing
4. Prepare equipment:
   - Router
   - Cables
   - Tools
   - Receipt book

### Installation Process

**Step 1: Customer Registration**
1. Click **Add Customer**
2. Enter details:
   ```
   Name: Customer full name
   Phone: Primary contact number
   Email: (optional)
   ID Number: National ID/Passport
   
   Address:
   - House/building number
   - Street name
   - Landmark
   - Area/Colony
   
   Service:
   - Package: Select from available
   - Installation Date: Today's date
   ```

**Step 2: Installation**
1. Visit customer location
2. Install equipment:
   - Position router properly
   - Connect cables
   - Ensure good signal
3. Take installation photos:
   - Router setup
   - Cable routing
   - Customer premises
4. Upload photos in app

**Step 3: Activation**
1. Complete installation in system
2. Click **Activate Service**
3. Test connection with customer
4. Train customer on usage

**Step 4: Payment**
1. Collect installation fee
2. Collect first month advance
3. Issue receipt
4. Record payment in system
5. Get customer signature on agreement

## Collecting Payments

### Payment Collection

**Using Mobile App**:
1. Open app
2. Search customer
3. Tap **Collect Payment**
4. Enter amount
5. Select method:
   - Cash
   - Mobile money
   - Bank transfer
6. Take photo of receipt/proof
7. Click **Submit**
8. SMS sent to customer

**Using Web**:
1. Go to customer profile
2. Click **Record Payment**
3. Fill payment details
4. Upload receipt
5. Save

### Payment Schedule

**Monthly Cycle**:
- Due Date: As per customer's billing date
- Grace Period: 3 days
- Reminder: Day before due date
- Suspension: Day 10 if unpaid

**Your Collection Target**:
- Aim for 95%+ collection rate
- Focus on payment dates
- Follow up consistently
- Maintain good relationships

### Cash Management

**Daily Process**:
1. Collect payments from customers
2. Record each payment immediately
3. Keep cash secure
4. Count at end of day
5. Deposit to Operator or bank
6. Get deposit receipt
7. Update system

**Best Practices**:
- Never delay recording payments
- Keep cash and personal money separate
- Deposit daily
- Maintain cash register
- Get receipts for all transactions

## Troubleshooting

### Customer Cannot Connect

1. **Check Status**:
   - Is customer active?
   - Payment up to date?
   - Service not suspended?

2. **Physical Check**:
   - Router powered on?
   - Cables connected?
   - Lights blinking correctly?

3. **Test**:
   - Can other devices connect?
   - Try factory reset router
   - Check username/password

4. **Escalate**:
   If not resolved, contact Operator with:
   - Customer details
   - Issue description
   - Steps already tried
   - Photos if relevant

### Slow Internet

1. Test speed on multiple devices
2. Check customer package speed
3. Verify bandwidth usage
4. Restart router
5. Check cables
6. Contact Operator if persists

### App Issues

**Cannot Login**:
- Check username/password
- Verify internet connection
- Clear app cache
- Update app
- Contact support

**Data Not Syncing**:
- Check internet
- Force sync
- Reopen app
- Wait and retry
- Contact support

## Frequently Asked Questions

**Q: How many customers can I manage?**  
A: Check your quota with Operator. Can be increased based on performance.

**Q: Can I create sub-operators?**  
A: No, only Operators can create sub-operators.

**Q: What if customer doesn't pay?**  
A: Follow payment policy. After grace period, suspend service. Report to Operator if needed.

**Q: Can I change package prices?**  
A: No, prices are set by Admin. Contact Operator for special cases.

**Q: What equipment do I need?**  
A: Coordinate with Operator for equipment supply.

**Q: Can I work in another area?**  
A: Only in your assigned area. Request area change from Operator if needed.
