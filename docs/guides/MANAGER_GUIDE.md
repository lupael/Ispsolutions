# Manager Guide - ISP Solution

## Role Overview

**Level**: 50  
**Access**: View/Edit permissions as granted by Admin

As a Manager, you have oversight permissions to view and potentially edit customer data, reports, and operations based on the specific permissions granted to you by the Admin.

## Getting Started

### Initial Login

1. Access the ISP Solution URL
2. Login with your credentials
3. Review your assigned permissions
4. Familiarize yourself with the dashboard

### Understanding Your Role

Your access depends on permissions granted by Admin:

**Typical Permissions**:
- View customer information
- View/edit specific modules
- Generate reports
- Monitor team performance
- View financial data (may be limited)
- Communication with customers

**You Cannot**:
- Create/delete users (unless explicitly permitted)
- Change system settings
- Access billing (unless permitted)
- Create service packages

## Customer Overview

### Viewing Customers

1. Navigate to **Customers**
2. View list with:
   - Customer name
   - Package
   - Status (Active/Suspended)
   - Last payment date
   - Assigned operator

### Customer Details

Click on customer to view:

**Account Tab**:
- Personal information
- Contact details
- Service package
- Connection status
- Operator assigned

**Billing Tab** (if permitted):
- Current balance
- Payment history
- Invoice list
- Transaction log

**Usage Tab**:
- Bandwidth consumption
- Session history
- Data usage patterns
- Peak usage times

**Support Tab**:
- Open tickets
- Ticket history
- Communication log
- Service notes

### Editing Customer Information

If you have edit permissions:
1. Open customer profile
2. Click **Edit**
3. Update allowed fields
4. Save changes

**Note**: Some fields may be read-only based on your permissions.

### Customer Search & Filters
