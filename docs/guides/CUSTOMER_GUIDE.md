### Common Issues & Solutions

#### Cannot Connect to Internet

**Check First**:
1. Is your account active?
2. Is payment up to date?
3. Is router powered on?
4. Are cables connected properly?

**Try**:
- Restart router (unplug for 30 seconds)
- Check username/password
- Try different device
- Check account status in portal

**Still not working?** Create support ticket.

#### Slow Internet

**Troubleshoot**:
1. Run speed test (speedtest.net)
2. Check your package speed
3. Try wired connection
4. Close heavy applications
5. Scan for viruses

**Multiple Devices**:
- Bandwidth shared among devices
- Heavy downloads affect speed
- Streaming uses more bandwidth

**Need Help?** Contact support.

#### Forgot Username/Password

**Recover**:
1. Click "Forgot Password" on login
2. Enter your email/phone
3. Follow reset instructions
4. Or contact support

## Connection Setup

### For PPPoE Connection

**Router Setup**:
1. Connect router to modem
2. Access router admin (usually 192.168.1.1)
3. Go to WAN/Internet settings
4. Select "PPPoE"
5. Enter your username
6. Enter your password
7. Save and connect

**Check Connection**:
- WAN status should show "Connected"
- You should get an IP address
- Try browsing

### For Hotspot Connection

**Connect to WiFi**:
1. Find WiFi network (provided by ISP)
2. Connect
3. Open browser
4. Enter username
5. Enter password
6. Click Login
7. Accept terms if shown

**Stay Connected**:
- Some hotspots require re-login
- Save credentials in browser
- Enable "Remember Me" if available

### For Static IP Connection

**Network Settings**:
1. Open Network Settings
2. Select "Manual" or "Static IP"
3. Enter details (provided by ISP):
   - IP Address
   - Subnet Mask
   - Gateway
   - DNS Servers
4. Save and apply

**Verify**:
- Check IP assigned correctly
- Try ping gateway
- Browse internet

## Tips & Best Practices

### Get Best Performance

**Router Placement**:
- Central location
- Away from walls/metal objects
- Elevated position
- Away from other electronics

**WiFi Security**:
- Use WPA2 encryption
- Strong WiFi password
- Change default admin password
- Disable WPS

**Optimize Usage**:
- Close unused applications
- Limit simultaneous downloads
- Schedule large downloads for off-peak
- Use wired connection for gaming/streaming

### Save Data

**Tips to Reduce Usage**:
- Lower video quality on streaming
- Disable auto-play videos
- Use data-saving browser modes
- Download instead of streaming when possible
- Monitor background app usage

### Security Tips

**Stay Safe Online**:
- Use strong passwords
- Don't share account credentials
- Keep devices updated
- Use antivirus software
- Be wary of phishing emails
- Use HTTPS websites
- Regular backup important data

### Payment Tips

**Avoid Service Interruption**:
- Pay before due date
- Set payment reminders
- Enable auto-payment if available
- Keep contact info updated
- Check email/SMS for reminders
