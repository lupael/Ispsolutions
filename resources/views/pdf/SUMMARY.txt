DESIGN SPECIFICATIONS
================================================================================

PAGE LAYOUT:
- Page Size: A4 (210mm x 297mm)
- Margins: 20mm all sides
- Font Family: System UI (optimal rendering)
- Base Font Size: 12px
- Line Height: 1.6

COLOR PALETTE:
- Primary: #007bff (Blue)
- Success: #28a745 (Green - Paid/Completed)
- Danger: #dc3545 (Red - Unpaid/Failed)
- Warning: #ffc107 (Yellow - Pending)
- Info: #17a2b8 (Cyan - Secondary)
- Light: #f8f9fa (Light Gray - Backgrounds)

STATUS INDICATORS:
- Invoice: Paid, Pending, Overdue, Cancelled, Draft
- Payment: Completed, Pending, Failed, Refunded
- Customer: Active, Inactive, Suspended, Cancelled, Pending

TYPOGRAPHY:
- H1: 24px, Bold, Primary color
- H2: 28px, Bold, Primary color

- Body: 12px, Regular
- Small: 11px, Regular
- Label: 12px, Bold, 600 weight

================================================================================
TENANT CONFIGURATION SUPPORT
================================================================================

Supported Settings Keys:
- company_logo_url: URL to company logo
- company_name: Company name (fallback to app name)
- company_address: Full company address
- company_phone: Company phone number
- company_email: Company email address
- invoice_terms: Default invoice terms and conditions

Example Configuration:
Tenant::create([
    'name' => 'Company Name',
    'settings' => [
        'company_logo_url' => 'https://example.com/logo.png',
        'company_address' => '123 Main Street, City, State 12345',
        'company_phone' => '(555) 123-4567',
        'company_email' => 'support@example.com',
        'invoice_terms' => 'Payment due within 30 days.',
    ],
]);
